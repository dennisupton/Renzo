<Renzo>

terminal-based HTML editor — no mouse, no bloat

$ python main.py index.html


Yes this website was actually writen in renzo lol

Features

  • Node tree view
  • Property inspector
  • Tag search
  • Live browser preview 127.0.0.1:5090
  • Save to file or clipboard Ctrl+S
  • 100% keyboard driven

Install

Requires Python 3.8+ 

                
                                                            pip install blessed flask pyperclip

Run on any HTML file or start from scratch: 

                
                                                            python main.py index.html    # load existing filepython main.py                # blank project (clipboard save)

Workflow example

Build a button with one class — live preview, no browser refresh.

  1. Select <body>
  2. Press n button Enter
  3. Press Enter
  4. Press n class="btn" Enter
  5. Press Esc i Click me Enter
  6. See changes live at 127.0.0.1:5090
  7. Press Ctrl S

How it works

Renzo parses HTML into a flat list of nodes + text objects, each with an indent convertToString() network.py
